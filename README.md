## 📝 과제 및 규칙

### 📍 과제
- 매 공유회 시작 10분 전까지 발표 자료 제출
    - 발표 자료의 형식 제한 없음 (노션, PPT 모두 가능)
- 대회 프로젝트 기획서, 참가 인증 등 제출은 운영진의 슬랙 공지에 따름

### ⚖️ 규칙
- 스터디 및 공유회는 기본적으로 **필참**, 불참 시 CT 경고 1회 부여
  - 지각 2회 누적 시 경고 1회 (시작 10분 후부터 지각 처리)
- 발표 자료 미제출 또는 미발표 시 해당 팀 **팀원 전체** 경고 1회 부여
- 결석 사유가 학교 공결 기준에 부합할 경우 사유서를 통해 경고 차감 가능
- 각 팀원은 전원 **최소 1회 발표** 필수
- 팀장은 해당 연도 트랙 경고 1회 차감 혜택 부여
- 1년의 CT 커리큘럼을 모두 참여한 경우 트랙 이수로 인정
